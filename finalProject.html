Final Project

This is my final project for CSC 106!
Me and my partner used the SCRUM process to design a
program that allows a user to use their mouse to color
in a few pictures that we drew using javascript.
The user can select from coloring in a dog, a bunny, a fish, or a pig.
Then, the user has the option to select either red, blue, green, or a random color option.
There is a clear button that will clear all drawings done by the user and also a home button if the user wishes to return home to the menu.
